# Fitigue

## Description
Fitigue is a full-stack clothing exchange and marketplace application. The system focuses on 
designing and implementing both frontend and backend components using proper software 
engineering principles. 
The frontend will provide an interactive and user-friendly interface where users can register, 
manage wardrobes, browse clothing listings, and initiate purchase or swap requests. The backend 
will handle business logic, user authentication, request processing, and communication with the 
database. 
The system will follow a layered architecture approach consisting of: 
- Presentation Layer (Frontend UI) 
- Business Logic Layer 
- Data Access Layer (Database Handling)
  
The project will emphasize modular design, requirement analysis, iteration-based development, 
role distribution, and integration between frontend and backend components. The main objective 
is to demonstrate a working full-stack application developed using structured software 
engineering practices.
